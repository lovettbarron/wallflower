    ```rust
    use sha2::{Sha256, Digest};
    use std::io::{BufReader, Read};
    use std::path::Path;
    use std::fs::File;
    use crate::error::Result;

    pub fn compute_sha256(path: &Path) -> Result<String> {
        let file = File::open(path)?;
        let mut reader = BufReader::with_capacity(8192, file);
        let mut hasher = Sha256::new();
        let mut buffer = [0u8; 8192];
        loop {
            let n = reader.read(&mut buffer)?;
            if n == 0 { break; }
            hasher.update(&buffer[..n]);
        }
        Ok(format!("{:x}", hasher.finalize()))
    }
    ```

    - `pub fn import_file(conn: &Connection, storage_dir: &Path, source: &Path) -> ImportResult` — the core import function:
      1. Validate file extension is .wav, .flac, or .mp3 (case-insensitive). Return Error variant if not.
      2. Compute SHA-256 hash via `hasher::compute_sha256(source)`.
      3. Check for duplicate via `db::find_by_hash(conn, &hash)`. If found, return `ImportResult::Duplicate { filename }` per D-01.
      4. Extract metadata via `metadata::extract(source)`.
      5. Get file size via `std::fs::metadata(source)?.len()`.
      6. Atomic copy per STOR-07 and D-01: create `NamedTempFile::new_in(storage_dir)`, copy source to temp via `std::io::copy`, call `temp.as_file_mut().sync_all()` (fsync), then `temp.persist(target_path)` (atomic rename).
      7. Target filename: use original filename, but if collision exists, append `-1`, `-2`, etc.
      8. Insert into database via `db::insert_jam(conn, &NewJam { ... })`.
      9. Return `ImportResult::Imported { filename, jam }`.
      10. On any error in steps 2-8, return `ImportResult::Error { filename, error: e.to_string() }`.
    - `pub fn import_files(conn: &Connection, storage_dir: &Path, paths: Vec<PathBuf>) -> Vec<ImportResult>` — map over paths, call `import_file` for each. Per D-04 (batch import).
