    3. Create crates/wallflower-app/src/commands/recording.rs with Tauri commands:

       `start_recording(state, app_handle)`:
       - Lock db, create a new jam record with `uuid::Uuid::new_v4()` as ID, filename = `{id}.wav`, format = "wav", imported_at = now. Insert into jams table.
       - Get storage_dir from config
       - Lock recording_engine, call `start(&storage_dir, &jam_id)`
       - Emit Tauri event "recording-started" with `{ jamId, jamName, deviceName }`
       - Return `{ jamId, deviceName }` (per D-10: frontend needs jam ID to navigate to detail view)

       `stop_recording(state, app_handle)`:
       - Lock recording_engine, call `stop()` to get file path
       - Update the jam record in DB: set duration_seconds, file_size_bytes, file_path from the finalized WAV
       - Compute content_hash (reuse existing SHA-256 hashing from import module)
       - Emit Tauri event "recording-stopped" with `{ jamId, filePath, durationSeconds }`
       - Return `{ jamId, filePath, durationSeconds }`

       `get_recording_status(state)`:
       - Lock recording_engine, call `status()` and `device_name()`
       - Return `{ state: String, deviceName: Option<String>, isRecording: bool }`

       `list_audio_devices()`:
       - Call `wallflower_core::recording::device::list_input_devices()`
       - Return `Vec<InputDeviceInfo>`

       `get_recording_level(state)`:
       - Drain the recording_event_rx for LevelUpdate events (non-blocking)
       - Return the most recent `rms_db` value or -100.0 if no updates
       - Also return any silence events that were pending

       ```rust
       use tauri_plugin_global_shortcut::{Code, GlobalShortcutExt, Modifiers, Shortcut, ShortcutState};

       // In Tauri Builder:
       .plugin(
           tauri_plugin_global_shortcut::Builder::new()
               .with_handler(move |app, shortcut, event| {
                   if event.state() == ShortcutState::Pressed {
                       // Toggle recording: check current state, start or stop
                       let state = app.state::<AppState>();
                       let engine = state.recording_engine.lock().unwrap();
                       match engine.status() {
                           RecordingState::Idle => {
                               // Start recording (same flow as Tauri command)
                               // Emit "recording-started" event

                           RecordingState::Recording => {

                           }
                           _ => {}
                       }
                   }
               })
               .build(),
       )

       // In setup closure:
       let shortcut = Shortcut::new(Some(Modifiers::META | Modifiers::SHIFT), Code::KeyR);
       app.global_shortcut().register(shortcut)?;
       ```
