       Add event listeners (inside useEffect, following existing pattern):

       a. "recording-level" event: `setLevel(payload.rmsDb)` -- updates the recording store with latest level data for waveform and meter rendering.

       b. "recording-state-changed" event: Handle state transitions:
          - If state === "device_disconnected": `setDeviceDisconnected(true)`, show persistent toast: "Audio device disconnected. Reconnect to resume recording."
          - If state === "recording" and was disconnected: `setDeviceDisconnected(false)`, toast: "Recording resumed -- {deviceName} reconnected" (3 seconds)
          - If state === "idle" and was recording: `reset()` (recording was stopped externally)
          - If state === "error": toast with error message (6 seconds)

       c. "recording-device-error" event: `setDeviceDisconnected(true)`. Native notification sent by backend.

       d. "recording-device-reconnected" event: `setDeviceDisconnected(false)`. Toast: "Recording resumed -- {deviceName} reconnected" (3 seconds).

       e. "recording-silence-start" event: Store the start offset temporarily (in a ref or local state). When "recording-silence-end" arrives, compute duration and call `addSilenceRegion(startSeconds, endSeconds)`. Convert sample offsets to seconds using sample rate (48000 default).

       f. "show-stop-dialog" event: Call `requestStop()` to open the StopRecordingDialog. Also bring the window to focus if not already focused.

       g. "recording-started" event (from backend after start): Could be used for synchronization if the UI was started via tray/hotkey. Show toast: "Recording from {deviceName}" (3 seconds, per UI-SPEC).

       h. "recording-stopped" event (from backend after stop): Show toast: "Recording saved -- {duration} captured" (4 seconds, per UI-SPEC).

    2. Add an elapsed time timer:
       - In the TauriEventListener component (or a new `useRecordingTimer` hook), start an interval when `isRecording` becomes true:
         ```typescript
         useEffect(() => {
           if (!isRecording) return;
           const startTime = Date.now();
           const interval = setInterval(() => {
             const elapsed = Math.floor((Date.now() - startTime) / 1000);
             setElapsed(elapsed);
           }, 1000);
           return () => clearInterval(interval);
         }, [isRecording, setElapsed]);
         ```
       - This provides the elapsed seconds that TransportBar and RecordingView display.
